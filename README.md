# Funcionário Digital — Nuxter

Protótipo de landing page para o produto **Funcionário Digital**: plataforma de funcionários digitais com IA (não chatbot), no estilo [Zaia](https://zaia.app/), com identidade visual **Nuxter** (preto e verde).

## Como rodar

Dê dois cliques em **`ABRIR-LANDING.bat`** — a página abre no seu navegador.

Ou abra o arquivo **`index-standalone.html`** diretamente no Chrome/Edge/Firefox. Não precisa de npm nem de instalar nada.

## Estrutura da landing

- **Hero**: headline “Contrate um Funcionário Digital”, CTAs (Testar grátis / Ver funcionários).
- **Problema**: por que automatizar em vez de contratar várias pessoas.
- **Funcionários**: cards (Financeiro, Cobrança, Compras, RH, Suporte interno).
- **Como funciona**: capacidades (lê e-mails, responde, gera solicitações, atualiza sistema, cobra, relatórios).
